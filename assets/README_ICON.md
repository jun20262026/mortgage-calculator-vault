# 应用图标说明

请用以下方式生成图标：

1. 打开 `icon-template.html` 在浏览器中
2. 截图保存为 `icon.png`（1024x1024 像素）
3. 同时创建以下尺寸的图标：
   - 1024x1024（App Store）
   - 180x180（iPhone）
   - 120x120（iPhone@2x）
   - 76x76（iPad）
   - 60x60（iPhone 设置页）

或者使用在线工具：
- https://appicon.co/
- https://icon.kitchen/

上传 icon.png 到 assets/ 目录即可。
